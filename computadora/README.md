# Proyecto: Simulador de Encendido de Computadora 💻

Este programa simula el encendido de una computadora verificando sus componentes internos y externos, además de considerar la postura del usuario para un uso correcto del equipo.

## Funcionalidades:
- Verificación de encendido/apagado.
- Verificación de componentes internos (CPU, RAM, etc.).
- Verificación de elementos externos (monitor, teclado, mouse).
- Validación de postura del usuario.

## Tecnologías usadas
- C#
- Programación orientada a objetos (clases abstractas, herencia, polimorfismo)
